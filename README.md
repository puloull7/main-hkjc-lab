# main-hkjc-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不依赖特定后端服务，可直接在浏览器中打开或通过 GitHub Pages 等静态托管服务访问。

仓库内容主要涉及前端展示、交互原型、实验性页面或信息归档用途，不针对任何具体网站或域名。

## 目录说明

```
main-hkjc-lab/
├── index.html           # 仓库首页 / 页面导航
├── pages/               # 存放各独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/              # 静态资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md            # 本文件
```

- `pages/` 目录下为每个独立页面，文件名可反映页面内容或用途。
- `assets/` 目录存放各页面共享或专用的资源文件，建议按类型分类存放。
- 根目录下的 `index.html` 可作为导航页，列出所有页面链接。

## 页面归档说明

每个 HTML 页面均为独立、完整的文件，可脱离仓库环境运行。页面之间无强依赖关系，便于单独引用或分发。

归档内容可能包括：

- 信息展示页
- 交互式工具原型
- 数据可视化示例
- 其他实验性或临时性页面

如需新增页面，请将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 中的导航链接（如有）。
